## [1.0.0] - 2026-04-30

### Cambios de marca
- Renombrado de "Smart Process CR" a "Smart Web CR" en toda la documentación
- Actualizado dominio a `smartwebcr.com` en sitemap.xml
- Actualizado email de contacto a `contacto@smartwebcr.com`

### Estructura del proyecto
- Catálogo principal con 6 plantillas profesionales
- Plantillas incluidas:
  - **ShopFlow** ($49) - E-commerce con home, catálogo y detalle de producto
  - **DevFolio Pro** ($29) - Portfolio para desarrolladores
  - **AgencyPro** ($39) - Landing page para agencias de marketing
  - **SaaSKit** (Gratis) - Landing page para productos SaaS
  - **Startup Hero** ($39) - Landing page para servicios profesionales
  - **CreativeCanvas** ($35) - Portfolio para creativos

### Características técnicas
- HTML5 semántico y accesible
- CSS3 con variables para fácil personalización
- JavaScript vanilla sin dependencias externas
- Dark/Light mode incluido en todas las plantillas
- Diseño 100% responsive (mobile-first)
- Optimización SEO (meta tags, sitemap.xml, robots.txt)
- Puntuación 90+ en Lighthouse

### Documentación
- README.md con estructura completa del proyecto
- GUIA_PERSONALIZACION.md con instrucciones detalladas
- CHANGELOG.md para seguimiento de versiones
- LICENSE.md con términos de uso
- sitemap.xml actualizado con todas las páginas
